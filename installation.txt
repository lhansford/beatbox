INSTALLATION

Beatbox needs a number of 3rd-party Python libraries in order to run. They are
listed below:

PySide - http://qt-project.org/wiki/PySide

Mutagen - http://mutagen.readthedocs.org/en/latest/#where-do-i-get-it

BeautifulSoup - 
http://www.crummy.com/software/BeautifulSoup/bs4/doc/#installing-beautiful-soup

Requests - http://docs.python-requests.org/en/latest/user/install/

Beatbox can play any music file supported by the OS running the
application. On Mac codecs to play Ogg Vorbis, Flac, and WMA files must be
installed seperately.

WMA codec for Mac - http://qtwmacodec.sourceforge.net/
FLAC/OGG codec for Mac - http://www.xiph.org/quicktime/
